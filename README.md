Project Overview-
This is a Real-Time Weather Dashboard that combines modern UI design with live data fetching. It uses a "Glassmorphism" style (frosted glass effect) to show weather conditions beautifully.

How it Works-
1. Live Data: It connects to the OpenWeatherMap API to get current weather for any city in the world.

2. Smart Date: I used JavaScript's Date object to ensure the day and month are always current. It doesn't just stay on one day; it updates every time the page loads.

3. Dynamic Icons: The app checks the weather status (Rain, Clouds, Clear) and automatically switches the central icon to match.

4. Modern CSS: Built with Bootstrap 5 for a responsive layout and custom CSS animations to make the weather icons "float."

 How to Use-
Type a city name (e.g., "Lucknow" or "azamgarh") into the search bar.

Press Enter or click the Search icon.

The card will instantly update with the temperature, wind speed, humidity, and pressure.

 Why I built this-
I created this project to practice Asynchronous JavaScript (API fetching) and to master CSS effects like backdrop-filter and absolute positioning.
